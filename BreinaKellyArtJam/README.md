## Description

My Art Jam project! This program will hopefully let users make their own art with trails of varying circles and lines!

Instructions:

Press 1 for circles
Press 2 for lines
Press 3 for nothing
Press 0 to clear the screen
Up and down arrow keys control shape size/line width
